# Portfolio Patrick Amavi

Portfolio professionnel moderne et animé construit avec React, TypeScript, Tailwind CSS et Framer Motion.

## Caractéristiques

- Design ultra moderne et responsive
- Animations fluides et transitions
- Section Hero avec effets visuels
- À propos avec mise en avant des compétences
- Galerie de compétences techniques avec barres de progression
- Projets professionnels (Cerconnet & AMAVGEST)
- Section UI/UX Works avec galerie d'images
- Témoignages clients avec système de gestion
- Formulaire de contact et soumission de témoignages
- Footer moderne avec liens de navigation
- Intégration Supabase pour la gestion des témoignages

## Technologies utilisées

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion (animations)
- React Icons
- Supabase (base de données)
- React Intersection Observer

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le site sera accessible sur `http://localhost:3000`

## Build

```bash
npm run build
```

## Sections du site

1. **Hero** - Introduction avec animation et liens sociaux
2. **À propos** - Présentation personnelle et compétences clés
3. **Compétences** - Technologies maîtrisées avec niveaux
4. **Projets** - Cerconnet et AMAVGEST avec descriptions détaillées
5. **UI/UX Works** - Galerie de créations design
6. **Témoignages** - Avis clients avec système de notation
7. **Contact** - Formulaire de contact et soumission de témoignages

## Configuration Supabase

Les variables d'environnement sont déjà configurées dans `.env`:
- `VITE_SUPABASE_URL` - URL du projet Supabase
- `VITE_SUPABASE_ANON_KEY` - Clé anonyme Supabase
